# Actor Functions

## Traders
- open a new position
- close an existing position
- early close an existing positions


## Liquidity providers
- initiate a deposit
- initiate withdrawals
- claim rewards
- compound rewards

## Stakers
- initiate a deposit
- initiate withdrawals
- claim rewards
- compound rewards

## Keepers
- open market and limit orders
- register the user
- submit the ```Permit``` signatures for USDC and ARM approvals
- close expired trades
- execute early close trades
- revokes approvals for USDC and ARB

## Admin
- adding and updating options markets
- pausing markets
- setting parameters
